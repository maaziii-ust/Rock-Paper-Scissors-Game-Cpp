# Rock Paper Scissors Game (C++)

## Overview

Rock Paper Scissors Game is a console-based game developed in C++. The player competes against the computer in multiple rounds, and the winner is determined based on the traditional Rock-Paper-Scissors rules.

## Features

* Player vs Computer Gameplay
* Random Computer Moves
* Multiple Rounds
* Score Tracking
* Match Result Display
* Interactive Console Interface

## Technologies Used

* C++
* Loops
* Switch Statements
* Conditional Logic
* Random Number Generation

## Game Rules

* Rock beats Scissors
* Scissors beat Paper
* Paper beats Rock
* The game consists of multiple rounds
* The player with the highest score wins the match
